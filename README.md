# Kerygma Landing Page

Landing page for [Kerygma](https://github.com/ukonuzoidx/kerygma-landing-page) — a real-time AI scripture detection system for live sermons.

## Local Development

Open `index.html` in your browser. No build tools required.

## Deployment

This is a static site. Deploy to GitHub Pages, Vercel, Netlify, or any static host.

To enable GitHub Pages:
1. Go to repo Settings → Pages
2. Set source to "Deploy from a branch"
3. Select `master` branch, `/ (root)` folder
4. Save
